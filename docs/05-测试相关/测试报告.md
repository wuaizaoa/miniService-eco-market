## 🎯 端到端业务流程

### 完整购物流程
1. **用户注册** → 调用 user-service 的 /api/user/register
2. **用户登录** → 调用 user-service 的 /api/user/login，获取JWT
3. **创建商品分类** → 调用 product-service 的 /api/product/category
4. **创建商品** → 调用 product-service 的 /api/product
5. **添加购物车** → 调用 order-service 的 /api/cart
6. **创建订单** → 调用 order-service 的 /api/order（Feign调用product-service扣库存）
7. **创建支付** → 调用 payment-service 的 /api/payment
8. **Mock支付** → 调用 payment-service 的 /api/payment/{id}/pay
9. **验证订单状态** → 调用 order-service 查询订单状态

## 🚀 启动与测试指南

### 前置条件
1. MySQL 已启动，数据库 lesson 已创建
2. Redis 已启动（用于购物车）
3. Nacos 已启动（服务注册中心）
4. JDK 1.8+ 已安装
5. Maven 3.x 已安装

### 启动步骤

#### 1. 编译项目
```bash
cd eco-market
mvn clean install -DskipTests
```

#### 2. 启动 user-service (8081)
```bash
cd user-service/user-service-start
mvn spring-boot:run
```

#### 3. 启动 product-service (8082)
```bash
cd product-service/product-service-start
mvn spring-boot:run
```

#### 4. 启动 order-service (8083)
```bash
cd order-service/order-service-start
mvn spring-boot:run
```

#### 5. 启动 payment-service (8084)
```bash
cd payment-service/payment-service-start
mvn spring-boot:run
```

### 运行测试

#### 1. 准备Python环境
```bash
cd api-tests
python -m venv venv
venv\Scripts\activate.bat
pip install -r requirements.txt
```

#### 2. 运行所有测试
```bash
pytest -v
```

#### 3. 运行端到端测试
```bash
pytest test_e2e_flow.py -v
```

#### 4. 生成HTML报告
```bash
pytest -v --html=reports/report.html --self-contained-html
```
