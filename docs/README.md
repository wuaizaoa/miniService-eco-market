## 项目简介
EcoMarket 是一个基于 Spring Boot + Spring Cloud Alibaba 的微服务电商项目，包含用户服务、商品服务、订单服务、支付服务等模块，采用 Docker 容器化部署。

## 开发环境
- Spring Boot 2.x
- Spring Cloud Alibaba (Nacos、Gateway、OpenFeign)
- MySQL 8.0
- Redis 6.x
- Docker 20.x
