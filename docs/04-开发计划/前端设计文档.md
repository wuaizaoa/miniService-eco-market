# Eco-Market 前端设计文档

## 一、项目概述

### 1.1 项目简介
Eco-Market 是一个以环保为主题的迷你购物电商网站前端项目，采用现代前端技术栈开发，为用户提供流畅的购物体验。

### 1.2 技术栈
- **框架**: React 18 + TypeScript + Vite
- **UI组件库**: Ant Design 5.x
- **路由**: React Router v6
- **状态管理**: Zustand
- **HTTP客户端**: Axios
- **样式方案**: Ant Design + CSS Modules

### 1.3 设计理念
- **主题**: 绿色环保，可持续发展
- **风格**: 现代、简洁、清新
- **目标**: 传递环保理念的同时提供优秀的购物体验

## 四、页面设计

### 4.1 路由配置

| 路径 | 页面 | 说明 | 权限 |
|------|------|------|------|
| `/` | 首页 | 展示推荐商品、分类导航 | 公开 |
| `/products` | 商品列表 | 商品筛选、排序、分页 | 公开 |
| `/products/:id` | 商品详情 | 商品信息、加入购物车 | 公开 |
| `/cart` | 购物车 | 购物车商品管理 | 需要登录 |
| `/checkout` | 订单确认 | 确认订单、选择支付方式 | 需要登录 |
| `/user/orders` | 订单列表 | 用户订单列表 | 需要登录 |
| `/user/orders/:id` | 订单详情 | 订单详情页 | 需要登录 |
| `/user` | 用户中心 | 个人信息 | 需要登录 |
| `/login` | 登录页 | 用户登录 | 公开 |
| `/register` | 注册页 | 用户注册 | 公开 |

### 4.2 核心页面说明

#### 首页 (`/`)
- 顶部导航：Logo、搜索框、购物车、用户中心
- 轮播图：促销活动展示
- 分类导航：商品分类快捷入口
- 商品展示：热门推荐、新品上市
- 底部页脚：关于我们、联系方式

#### 商品列表页 (`/products`)
- 分类筛选侧边栏
- 排序功能（价格、销量、新品）
- 商品卡片网格布局
- 分页组件

#### 商品详情页 (`/products/:id`)
- 商品图片轮播
- 商品基本信息（名称、价格、库存）
- 商品描述
- 操作按钮：加入购物车、立即购买

#### 购物车页 (`/cart`)
- 购物车商品列表
- 数量增减、删除功能
- 总价计算
- 结算按钮

#### 订单确认页 (`/checkout`)
- 收货地址（模拟）
- 商品清单
- 支付方式选择（支付宝/微信）
- 提交订单按钮

#### 用户中心 (`/user`)
- 个人信息展示
- 订单列表入口

## 五、API对接

### 5.1 服务端口
根据后端微服务架构设计：
- user-service: 8081
- product-service: 8082
- order-service: 8083
- payment-service: 8084

### 5.2 统一响应格式
```typescript
interface ApiResponse<T> {
  code: number;
  message: string;
  data: T;
}
```

### 5.3 主要API接口

#### 用户服务
- `POST /api/user/register` - 用户注册
- `POST /api/user/login` - 用户登录
- `GET /api/user/info` - 获取用户信息

#### 商品服务
- `GET /api/product/list` - 获取商品列表
- `GET /api/product/:id` - 获取商品详情
- `GET /api/product/categories` - 获取分类列表

#### 订单服务
- `GET /api/order/cart` - 获取购物车
- `POST /api/order/cart` - 添加购物车
- `PUT /api/order/cart/:productId` - 更新购物车
- `DELETE /api/order/cart/:productId` - 删除购物车商品
- `POST /api/order/create` - 创建订单
- `GET /api/order/list` - 获取订单列表
- `GET /api/order/:id` - 获取订单详情

#### 支付服务
- `POST /api/payment/pay` - 支付订单

## 六、状态管理

### 6.1 用户状态 (useUserStore)
```typescript
interface UserState {
  user: User | null;
  token: string | null;
  isLoggedIn: boolean;
  login: (user: User, token: string) => void;
  logout: () => void;
}
```

### 6.2 购物车状态 (useCartStore)
```typescript
interface CartItem {
  productId: number;
  productName: string;
  productPrice: number;
  quantity: number;
  image?: string;
}

interface CartState {
  items: CartItem[];
  totalCount: number;
  totalPrice: number;
  addItem: (item: CartItem) => void;
  updateQuantity: (productId: number, quantity: number) => void;
  removeItem: (productId: number) => void;
  clearCart: () => void;
}
```

### 6.3 商品状态 (useProductStore)
```typescript
interface ProductState {
  products: Product[];
  categories: Category[];
  currentProduct: Product | null;
  loading: boolean;
  setProducts: (products: Product[]) => void;
  setCategories: (categories: Category[]) => void;
  setCurrentProduct: (product: Product | null) => void;
}
```
