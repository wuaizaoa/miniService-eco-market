# Eco-Market 前端实施计划

> **说明:** 本计划包含 Eco-Market 前端项目的完整实施步骤，按照任务分解逐步实现。

**目标:** 构建一个完整的环保主题电商平台前端，包含用户、商品、购物车、订单、支付等核心功能模块。

**架构:** 采用 React + TypeScript + Vite + Ant Design + Zustand 的单页应用架构，前后端分离，通过 RESTful API 与后端微服务通信。

**技术栈:** React 18, TypeScript, Vite, Ant Design 5, React Router v6, Zustand, Axios
