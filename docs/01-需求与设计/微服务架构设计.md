```
eco-market/
├── eco-market-common/              # 公共模块

│   ├── user-service-adapter/       # 适配层（Controller）

├── payment-service/                # 支付服务（结构同user-service）
└── pom.xml                         # 父工程POM
```

## 二、COLA 4 架构分层说明

### 2.1 分层依赖规则

```
adapter → app → domain ← infrastructure
          ↓
        client
          ↑
        start
```

- **domain（领域层）**: 核心，不依赖任何其他层
- **infrastructure（基础设施层）**: 依赖 domain，实现 Repository 接口
- **app（应用层）**: 依赖 domain，编排业务逻辑
- **adapter（适配层）**: 依赖 app，对外暴露 REST API
- **client（API层）**: 供其他服务调用的 Feign Client 接口定义
- **start（启动层）**: Spring Boot 启动类、配置文件

### 2.2 各层职责

| 层级 | 职责 | 示例 |
|------|------|------|
| domain | Entity、Domain Service、Repository 接口 | User、UserRepository |
| infrastructure | Repository 实现、Mapper、外部调用 | UserMapperImpl、UserMapper |
| app | Application Service、Command、Query | UserService、RegisterCmd |
| adapter | Controller、请求响应转换 | UserController |
| client | Feign Client 接口定义 | UserFeignClient |
| start | Spring Boot Application、配置 | UserApplication、application.yml |

## 七、各服务模块职责

### 7.1 user-service（用户服务）
- 用户注册、登录
- 用户信息查询与修改
- JWT Token 生成与验证

### 7.2 product-service（商品服务）
- 商品分类管理
- 商品 CRUD
- 库存扣减（供订单服务调用）

### 7.3 order-service（订单服务）
- 购物车管理（Redis）
- 订单创建与查询
- 订单状态流转
- 调用商品服务扣减库存

### 7.4 payment-service（支付服务）
- 支付记录创建
- Mock 支付实现
- 调用订单服务更新订单状态
